I wasn't sure if [[keeping externally-generated material directly in my notes]] was a good idea or not because I worried about [[dim - assimilation depth -- low]].

I'm less concerned now, as long as:
* The writer wrote something _close enough_ to how I would write it
* The note is short and conceptual
* I *completely* understand it when I first read it, and could explain it back to someone nearly verbatim
* I keep an annotation about the original authorship.  Often I just [[tag page]] it with "[[AI says]]"

Counter example: you might think it would be possible to pull in [[Andy's note about evergreen notes functioning like spaced repetition]] since it is short and conceptual, but that falls apart in the "writes something _close enough_ to how I would write it" and "completely understand it when I first read it" areas.

That _probably_ means that for [[externally-generated material]] it might make sense to have some kind of a [[source conversion for assimilation of externally-generated material]], like with an [[agent]] at the boundary.
